Smart Bus 🚍 Route Finder (Path finding System) 🗺️⚡🔎

A Prolog-based route-finding system designed to calculate optimal paths between cities using multiple search algorithms—BFS, DFS, and A*. The system integrates real bus route data, includes safety checks for unsuitable inputs, and is structured for scalability and flexibility.

Extensive testing and refinement ensured accuracy, stability, and reliability across different scenarios. The project demonstrates how an academic concept was transformed into a practical, usable tool. 💡
